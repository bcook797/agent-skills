# Agent Skills

A collection of installable skills that extend AI agents with specialized architectural and engineering workflows. Works with Claude Code, OpenCode, Cursor, Copilot, Cline, Windsurf, Gemini, and [many more](https://skills.sh).

## Skills

### Architecture & Design

**adr-builder** — Guides the creation of Architecture Decision Records (ADRs) following [Michael Nygard's format](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions). Covers gathering context, drafting, quality-checking, and writing the ADR file via [adr-tools](https://github.com/npryce/adr-tools).

```
npx skills@latest add bcook797/skills/adr-builder
```

---

**nfr-architect** — Guides teams through writing, validating, and operationalizing Non-Functional Requirements (NFRs). Translates requirements into SLOs and SLIs, and generates automatable fitness functions (performance tests, ArchUnit-style architecture tests, availability checks, recovery drills).

```
npx skills@latest add bcook797/skills/nfr-architect
```

---

### Strategy & Communication

**tech-radar-blip** — Guides the creation of compelling, well-documented blip submissions for the [ThoughtWorks Tech Radar](https://www.thoughtworks.com/radar). Covers quadrant and ring selection, evidence gathering, and drafting a submission ready for editorial review.

```
npx skills@latest add bcook797/skills/tech-radar-blip
```

---

**writing-assistant** — Provides structured writing feedback for Articles, Conference Talk Submissions, and Short Posts. Evaluates style (active voice, conciseness, American English), clarity, narrative coherence, and formatting.

```
npx skills@latest add bcook797/skills/writing-assistant
```
